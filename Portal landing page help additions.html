Internet Explorer 11.x

General

Internet Explorer allows a site-specific setting for pop-ups, but does not allow a site-specific setting for downloads. For optimal security, we recommend visiting the Stewardship Portal via Google Chrome, as Chrome allows both settings to be site-specific.

Desktop Settings

Click the Tools button (gear at the top right of screen), then select "Internet Options". Check these three settings:
  1. Enable file downloads.
  2. Allow popups on this site.
  3. Allow JavaScript (we assume this is already enabled, or you're missing out on most of the web).
  On the "Security" tab, click "Custom Level" and the dialog box at right should appear. Click "enable" under file downloads. This will allow you to download PDFs from the Stewardship Portal. Farther down on the same list, under "Scripting", under "Active Scripting", click "enable" to allow JavaScript."
On the "Privacy" tab, click you can keep the Pop-up Blocker turned on, but click "Settings", then the dialog box at right appears. In that box, type "n413.fmphost.com" under "Address of website to allow", then click "Add". Then the site should appear in the "Allowed sites" box below. The Stewardship Portal utilizes pop-ups for two actions: viewing PDFs interactively, and linking to Furbish's website pages about coverage rates, thrive rates, and weed pressure.

Google Chrome 48 or higher

General

We recommend visiting the Stewardship Portal via Google Chrome, as Chrome allows you the most flexibility via site-specific security settings, and works equally well on PCs and Macs.

Desktop Settings

While on this site, click on the green lock to the left of the URL. The settings are the same on a PC or Mac desktop. Check these three settings:
  1. Allow automatic downloads on this site.
  2. Allow popups on this site.
  3. Allow JavaScript (we assume this is already enabled, or you're missing out on most of the web).
The Stewardship Portal utilizes pop-ups for two actions: viewing PDFs interactively, and linking to Furbish's website pages about coverage rates, thrive rates, and weed pressure. If you do not wish to enable pop-ups, you can still download PDFs from the Stewardship Portal.

Safari 9.x (on Mac)

General

Sarfari generally does not allow site-specific settings. If you have security concerns about the recommended settings for Safari, we recommend that you use Google Chrome, as Chrome allows site-specific settings.  

Desktop Settings

Click the "Safari" menu. Under the "Security" tab, un-check "Block pop-up windows". Also make sure JavaScript is enabled. Please note, this will allow pop-ups for all sites that you visit.
  The Stewardship Portal utilizes pop-ups for two actions: viewing PDFs interactively, and linking to Furbish's website pages about coverage rates, thrive rates, and weed pressure. If you do not wish to enable pop-ups, you can still download PDFs from the Stewardship Portal.
