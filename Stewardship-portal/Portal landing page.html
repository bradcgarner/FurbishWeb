Introducing the Furbish Stewardship Portal

Introduction

The Stewardship Portal is a web app for Furbish’s clients. After each Stewardship visit, we post photographic reports, which the Portal compiles as a project history. We also include other information such as the crew leader and health metrics of our clients’ living roofs and living walls.

Each client can view – and comment on – reports and information about their projects. Clients are only able to view their own projects. If you are a Furbish client, and we have not sent you an invitation to the Portal, please Contact Us.

General Help

We recommend visiting the Stewardship Portal via Google Chrome, as Chrome allows you the most flexibility via site-specific security settings, and works equally well on PCs and Macs. Be sure to allow JavaScript (we assume this is already enabled, or you're missing out on most of the web).

PDFs via Google Chrome on a Mac Device

The Stewardship Portal uses PDFs to display reports prior to 2017.

We recommend that you use Google Chrome's built-in PDF viewer to see report details in full within the Portal. Using Adobe Acrobat will allow you to download PDFs but does not allow you to view PDFs interactively within the Portal.

Follow the steps below to use Chrome's built-in PDF viewer:

  1. At the top right, click on the More icon and then Settings.
  2. At the bottom, click Show advanced settings.
  3. Under "Privacy," click Content settings.
  4. Under "PDF Documents", uncheck the box next to "Open PDF files in the default PDF viewer application."
  5. Chrome will now open PDFs automatically when viewed within the Portal.

PDFs via Microsoft Edge on a Windows PC

The Stewardship Portal uses PDFs to display reports prior to 2017.

Microsoft Edge includes a built-in PDF reader so you don't need to download a separate PDF viewer program.

Follow the instructions below to set Microsoft Edge as your default program for viewing PDFs:
  1. Click on the Start - Settings - Default Apps (or simply type 'Default Apps')
  2. Select Choose default apps by file type at the bottom of the screen
  3. Scroll down until you see the .pdf option and select Microsoft Edge as the default app.

Web Links via Google Chrome on a Mac Device

The Stewardship Portal uses web links to display information related to coverage rates, thrive rates, and weed pressure, which are located on Furbish's website, outside the Portal.

We recommend that you select the "Stay inside the Portal" option in your Web Link Preferences for optimal performance. This will open all web links in the same window without leaving the Stewardship Portal.


Choosing the "Use popups" option will require you to allow pop-ups for this website within your browser settings. If you decide to use pop-ups, we recommend you allow pop-ups for the Stewardship Portal specifically, not all websites.

There are two ways to allow pop-ups for the Stewardship Portal:

Option 1: Allowing pop-ups for the Stewardship Portal specifically (after clicking an external web link)

You should receive an error message (similar to the image below) blocking the pop-up the first time you try visiting a web link outside of the Portal.

Follow the steps below to always allow pop-ups for the Stewardship Portal:

  1. In the address bar, click Pop-ups blocked icon .
  2. Select Always show pop-ups from http://stewardship.furbishco.com and then Done.

Option 2: Allowing pop-ups specifically for the Stewardship Portal from the settings menu
  1. At the top right, click on the More icon and then Settings.
  2. At the bottom, click Show advanced settings.
  3. Under "Privacy," click Content settings.
  4. Under "Pop-ups", click the "Manage exceptions..." option
  5. Add "stewardship.furbishco.com" to the list of allowed pop-up exceptions
